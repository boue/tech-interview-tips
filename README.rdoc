== Technical Interview Tips

=== Setup:
- Ruby version: 2.1.1
- Rails version: 4.0.3
- Add config/database.yml file for postgres
- Add these to your ~.bash_profile file:
     export GITHUB_KEY="Shhhhh...Secret"
     export GITHUB_SECRET="Shhhhh...Secret"
- Run: 
    $bundle
    $rake db:setup
